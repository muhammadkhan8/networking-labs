# Lab 5 — Router-on-a-Stick

## Objective
Configure Router-on-a-Stick inter-VLAN routing using router subinterfaces and 802.1Q trunking.

## Skills Demonstrated
- Inter-VLAN routing
- Router subinterfaces
- 802.1Q encapsulation
- VLAN trunking
- Connectivity testing
- Cisco CLI troubleshooting

## Technologies Used
- Cisco Packet Tracer
- Cisco 1941 Router
- Cisco 2960 Switch
- VLANs
- IPv4 Addressing
- Cisco IOS
