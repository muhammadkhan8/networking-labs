# Lab 2 — Inter-VLAN Routing

## Objective
Configure VLANs and enable communication between separate networks using inter-VLAN routing.

## Skills Demonstrated
- VLAN configuration
- Access port assignment
- Trunk port configuration
- Router subinterfaces
- 802.1Q encapsulation
- Connectivity testing

## Technologies Used
- Cisco Packet Tracer
- Cisco 2960 Switch
- Cisco 1941 Router
- IPv4 Addressing
