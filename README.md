# Networking Labs

This repository contains hands-on networking labs completed using Cisco Packet Tracer.

## Focus Areas
- VLAN configuration
- Inter-VLAN routing
- Trunking
- SSH remote management
- IPv4/IPv6 addressing
- Cisco CLI
- Network troubleshooting
- Enterprise network fundamentals

## Lab List
- Lab 01 — VLAN Basics
- Lab 02 — Inter-VLAN Routing
- Lab 03 — Multi-Switch Enterprise Network
- Lab 04 — SSH Remote Management
- Lab 05 — Router-on-a-Stick

## Purpose
These labs were created to build hands-on networking, troubleshooting, and IT support skills for help desk, desktop support, and network support roles.
