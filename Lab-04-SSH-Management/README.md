# Lab 4 — SSH Remote Management

## Objective
Configure secure remote management access on Cisco network devices using SSH.

## Skills Demonstrated
- SSH configuration
- Local user authentication
- RSA key generation
- Secure remote access
- Cisco CLI configuration
- Basic switch security

## Technologies Used
- Cisco Packet Tracer
- Cisco 2960 Switch
- SSH Version 2
- IPv4 Addressing
- Cisco IOS
