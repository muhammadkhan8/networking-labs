# Lab 3 — Multi-Switch Enterprise Network

## Objective
Configure a multi-switch VLAN enterprise network with trunking, management VLANs, and SSH remote access.

## Skills Demonstrated
- VLAN configuration
- Trunk configuration
- SSH remote access
- Switch management VLANs
- Local authentication
- Cisco CLI troubleshooting

## Technologies Used
- Cisco Packet Tracer
- Cisco 2960 Switches
- VLANs
- 802.1Q Trunking
- SSH Version 2
- IPv4 Addressing
